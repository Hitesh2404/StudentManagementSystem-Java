# Student Management System (Java)

A simple Java console-based Student Management System demonstrating OOP concepts, ArrayList usage, and menu-driven programming.

## Features
- Add Student
- View All Students
- Search Student by ID
- Delete Student
- Exit Program

## Technologies Used
- Java
- OOP (Classes, Objects, Methods)
- ArrayList for dynamic data storage
- Scanner for user input
